# Submit Progress — Unit Converter for Science

**产品**: [Unit Converter for Science](https://www.unitconv.top)  
**目标**: 免费外链提交进度管理  
**规则**: 每次提交间隔 1-3 分钟，每日最多 10 个

---

## 状态说明

| 标记 | 含义 |
|------|------|
| ⬜ 待提交 | 尚未操作 |
| 🔄 提交中 | 已打开表单，等待审核 |
| ✅ 已完成 | 提交成功/审核通过 |
| ❌ 失败 | 提交失败（备注原因） |
| ⏭ 跳过 | 不适合/已关闭 |

## 提交命令参考

```bash
# 检查已提交记录
node src/cli.js status

# 提交到指定站点（已有 adapter）
node src/cli.js submit futuretools --engine bb

# 提交到任意 URL（通用 adapter）
node src/cli.js submit https://toolscout.ai/submit --engine bb

# 启动 Chrome（首次需要）
bb-browser open about:blank
```
